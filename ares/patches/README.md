# 设备专用补丁目录

将 K40G (ares) 设备专用的内核补丁放在这里。

## 补丁命名规范
```
0001-描述.patch    # 基础修复
0002-描述.patch    # 功能添加
...
```

## 常用补丁示例

### 1. MTK 平台兼容性补丁
- 修复 kprobe 问题
- 修复 mtk_customize.h 冲突

### 2. 驱动修复补丁
- WiFi 驱动修复
- 触控驱动修复
- 充电驱动修复

### 3. 性能优化补丁
- 调度器优化
- 内存管理优化

## 注意事项
1. 补丁按序号顺序应用
2. 确保补丁与内核版本（4.14.186）匹配
3. 应用失败的补丁会跳过并输出警告
